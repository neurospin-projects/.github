## Welcome to neurospin-projects 👋

- 🙋‍♀️ A short introduction - neurospin-projects provides a centralized location for hosting analysis code in a “paper‑with‑code” style,
  making it easier to share, reproduce, and validate analysis of neuroimaging data.
- neurospin-projects provides a centralized location to host your code in a "paper with code" fashion to facilitate reproducible analysis of neuroimaging data.
